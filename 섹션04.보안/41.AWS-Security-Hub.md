## AWS Security Hub 개요

AWS Security Hub는 전체 AWS 계정에 걸쳐 **보안 상태를 중앙에서 관리**하고, **보안 검사를 자동화**하는 통합 보안 서비스입니다.

* **중앙 집중화:** 여러 AWS 서비스와 파트너 도구의 경고 알림(Findings)을 하나의 대시보드에 집계합니다.
* **규정 준수:** 현재 리소스들이 보안 표준 및 규정을 잘 지키고 있는지 한눈에 파악하고 빠른 조치를 취할 수 있게 돕습니다.

---

## 주요 기능 및 통합 서비스

### 1. 보안 결과(Findings) 집계

Security Hub는 다음과 같은 다양한 AWS 서비스 및 파트너 솔루션으로부터 데이터를 수집합니다.

* **보안 서비스:** Amazon GuardDuty, Amazon Inspector, Amazon Macie
* **관리 및 분석:** AWS Config, IAM Access Analyzer, AWS Systems Manager, AWS Firewall Manager
* **상태 및 지원:** AWS Health, AWS Partner Network(APN) 솔루션

### 2. 자동화된 보안 검사

* 설정된 보안 기준에 따라 리소스를 자동으로 스캔합니다.
* **필수 조건:** Security Hub가 제대로 작동하려면 **AWS Config가 반드시 활성화**되어 있어야 합니다.

### 3. 사후 조치 및 조사 연동

* **Amazon EventBridge:** 보안 이슈 발생 시 이벤트를 생성하여 자동화된 대응(Lambda 호출 등)을 수행할 수 있습니다.
* **Amazon Detective:** 보안 이슈의 근본 원인을 빠르고 심층적으로 조사할 때 연동하여 사용합니다.

---

## Security Hub 활성화 단계

강의에서 설명한 Security Hub 설정 프로세스는 다음과 같습니다.

1. **AWS Config 활성화:** 리소스 기록 및 준수 여부 확인을 위한 선행 작업입니다.
2. **보안 기준(Standards) 선택:** 다음 3가지 주요 표준 중 준수할 항목을 선택합니다.
   * *AWS Foundational Security Best Practices*
   * *CIS AWS Foundations Benchmark*
   * *PCI DSS*


3. **서비스 통합:** 활성화된 기준에 따라 다양한 서비스의 결과를 통합합니다.
4. **Security Hub 활성화:** 최종적으로 버튼을 클릭하여 대시보드를 생성합니다.

## 요금 체계 및 참고 사항

* **가격 책정:**
  * **보안 검사 횟수:** 검사 건당 비용이 발생하며, 구간별로 단가가 달라집니다.
  * **이벤트 수집(Event Ingestion):** 첫 10,000건은 무료이며, 이후 건당 비용이 발생합니다.


* **체험판:** 30일간 무료로 체험해 볼 수 있는 프리 티어를 제공합니다.

---

> **핵심 요약**
>
> Security Hub는 **"수집(Config/GuardDuty 등) -> 집계(Hub 대시보드) -> 대응(EventBridge/Detective)"**으로 이어지는 AWS 보안 관리의 컨트롤 타워 역할을 합니다.

## ## 1. Amazon SageMaker 개요

**Amazon SageMaker**는 개발자와 데이터 과학자가 머신러닝(ML) 모델을 쉽고 빠르게 **구축, 학습, 배포**할 수 있도록 지원하는 **완전 관리형 서비스**입니다.

* **기존 AI 서비스와의 차이점:** * Rekognition, Polly 등 기존 서비스가 특정 기능(이미지 분석, 음성 변환 등)을 바로 사용하는 것이라면, SageMaker는 사용자가 **직접 커스텀 모델을 제작**할 때 사용합니다.
* **필요성:** * ML 모델 구축은 과정이 복잡하고 개별 단계마다 서버를 프로비저닝해야 하는 번거로움이 있습니다. SageMaker는 이 모든 과정을 한 곳에서 관리하게 해줍니다.

---

## ## 2. 머신러닝 모델 구축 과정 (Simple Example)

강의에서 언급된 'AWS 시험 점수 예측' 모델을 예시로 단계별 과정을 살펴보겠습니다.

### **단계 1: 데이터 수집 및 라벨링 (Labeling)**

* **설명:** 모델 학습을 위한 기초 데이터를 모으고 의미를 부여하는 단계입니다.
* **예시:** 학생 10,000명의 데이터(IT 경력, AWS 사용 기간, 학습 시간, 모의고사 횟수 등)를 수집하고, 각 데이터에 '실제 시험 점수'라는 **라벨(정답)**을 붙입니다.

### **단계 2: 모델 구축 및 학습 (Build & Train)**

* **설명:** 수집된 과거 데이터를 바탕으로 패턴을 파악하여 예측 모델을 만듭니다.
* **SageMaker의 역할:** 복잡한 인프라 설정 없이 머신러닝 알고리즘을 선택하고 학습시킬 수 있는 환경을 제공합니다.

### **단계 3: 튜닝 (Tuning)**

* **설명:** 학습된 모델이 실제 결과와 더 잘 맞도록 매개변수를 조정하여 정확도를 높이는 과정입니다.

### **단계 4: 배포 및 추론 (Deploy & Inference)**

* **설명:** 완성된 모델을 실제 서비스에 적용하여 새로운 데이터를 예측(추론)하는 단계입니다.
* **예시:** 새로운 수강생이 자신의 정보를 입력하면, 모델이 "이 학생은 906점으로 합격할 것 같습니다"라고 결과를 내놓습니다.

## ## 3. SageMaker의 핵심 가치 요약

| 단계 | 주요 기능 | 설명 |
| --- | --- | --- |
| **Labeling** | Ground Truth | 복잡한 데이터 라벨링 작업을 간소화 |
| **Build** | Notebooks | 모델 코드를 작성할 수 있는 환경 제공 |
| **Train & Tune** | Training Jobs | 서버 관리 없이 대규모 학습 및 하이퍼파라미터 최적화 |
| **Deploy** | Endpoints | 생성된 모델을 API 형태로 배포하여 즉시 사용 |

> **핵심 요약:** SageMaker는 라벨링부터 모델 구축, 학습, 튜닝, 그리고 실제 배포에 이르기까지 **머신러닝의 전체 라이프사이클**을 한 곳에서 해결해 주는 서비스입니다.

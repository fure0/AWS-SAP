# 🚀 Amazon Rekognition 핵심 정리

Amazon Rekognition은 머신러닝 기술을 사용하여 **이미지 및 동영상 분석을 자동화**하는 완전 관리형 서비스입니다. 별도의 딥러닝 지식 없이도 API 호출만으로 강력한 시각 분석 기능을 애플리케이션에 통합할 수 있습니다.

---

## 1. 주요 기능 및 활용 사례

Rekognition은 단순히 사물을 식별하는 것 이상의 정교한 분석 기능을 제공합니다.

* **라벨링 (Object & Scene Detection):** 사물(바위, 자전거), 장면(산꼭대기, 야외), 개념(골든 리트리버) 등을 식별하여 태그를 생성합니다.
* **얼굴 분석 (Facial Analysis):** 성별, 연령대, 감정(행복 등), 안경 착용 여부 등을 분석합니다.
* **얼굴 검색 및 비교 (Face Search & Verification):** 데이터베이스의 인물과 비교하여 신원을 확인합니다. (보안 및 사용자 인증)
* **텍스트 감지 (Text Detection):** 이미지 내의 텍스트를 추출합니다. (예: 마라톤 선수의 배번호 인식)
* **유명인 인식 (Celebrity Recognition):** 수천 명의 유명인 데이터베이스와 대조하여 인물을 찾습니다.
* **경로 찾기 (Pathing):** 동영상 내에서 사람이나 객체의 이동 동선을 실시간으로 분석합니다. (예: 축구 경기 분석)

---

## 2. 콘텐츠 조정 (Content Moderation)

**시험에서 특히 중요하게 다뤄지는 기능**으로, 안전한 사용자 경험을 위해 부적절한 콘텐츠를 걸러내는 프로세스입니다.

### 🛠 작동 방식

1. **이미지 분석:** Rekognition이 이미지를 스캔하여 부적절한 요소(인종 차별, 포르노 등)가 있는지 확인합니다.
2. **신뢰 임계값(Confidence Threshold) 설정:**
   * Rekognition이 해당 콘텐츠가 부적절하다고 확신하는 정도(%)를 설정합니다.
   * **임계값을 낮게 설정할수록:** 더 엄격하게 필터링되어 더 많은 항목에 플래그(Flag)가 지정됩니다.

3. **수동 검토 (Human-in-the-loop):**
   * 시스템이 자동으로 판단하기 모호한 경우, 사람이 직접 검토하도록 워크플로우를 구성합니다.

### 🤖 Amazon A2I (Augmented AI) 연동

* 자동화된 머신러닝 분석 결과 중 신뢰도가 낮은 항목을 **사람이 직접 수동 검토**할 수 있게 해주는 서비스입니다.
* 최종적으로 이미지를 보관할지, 삭제할지 결정하여 규정 준수 및 서비스 운영 정책을 유지합니다.

## 💡 시험 대비 핵심 포인트 (Summary)

| 구분 | 주요 특징 |
| --- | --- |
| **입력 데이터** | 이미지(Image) 및 동영상(Video) |
| **핵심 키워드** | 얼굴 인식, 객체 탐지, 텍스트 추출, **콘텐츠 조정** |
| **비즈니스 가치** | 자동화된 미디어 분석, 보안 강화, 부적절한 콘텐츠 차단 |
| **인적 검토** | **Amazon A2I**를 통해 수동 검토 프로세스 결합 가능 |

> **Tip:** 시험 문제에서 "사용자가 업로드한 이미지 중 부적절한 콘텐츠를 자동으로 감지하고, 필요한 경우 사람이 검토하는 아키텍처"를 묻는다면 **Amazon Rekognition + Amazon A2I** 조합을 기억하세요!

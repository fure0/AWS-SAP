## ## 🔍 Amazon Kendra: 머신러닝 기반 지능형 검색 서비스

Amazon Kendra는 단순한 키워드 매칭을 넘어, 머신러닝(ML)을 통해 기업 내 방대한 문서에서 **정확한 답변을 찾아내는 지능형 검색 서비스**입니다.

### 1. 주요 특징 및 기능

* **완전 관리형 서비스:** 인프라 관리 없이 바로 사용할 수 있는 ML 기반 문서 검색 서비스입니다.
* **문서 내 정답 추출:** 단순 문서 링크만 제공하는 것이 아니라, 문서의 내용을 이해하고 직접적인 답을 추출합니다.
* **자연어 질의(Natural Language Search):** 구글 검색처럼 일상적인 언어로 질문할 수 있습니다.
* *예: "IT 지원 데스크가 어디야?" → 결과: "1층"*


* **다양한 문서 형식 지원:**
* 텍스트, PDF, HTML, 파워포인트(PPT), MS 워드(Word), FAQ 등.



### 2. 데이터 소스 및 색인(Indexing)

* **방대한 연결성:** AWS 내외부의 다양한 데이터 소스를 연결하여 통합 검색이 가능합니다.
* **지식 색인 구축:** 연결된 데이터 소스를 스캔하여 내부적으로 머신러닝 기반의 지식 색인을 생성합니다.

### 3. 검색 결과 최적화 (ML 기반 학습)

* **점진적 학습 (Incremental Learning):** 사용자의 검색 클릭 패턴이나 피드백을 통해 어떤 결과가 유용한지 스스로 학습하고, 선호되는 결과를 상단에 추천합니다.
* **미세 조정 (Fine-Tuning):** 관리자는 특정 기준에 따라 검색 결과의 우선순위를 조정할 수 있습니다.
* 데이터의 **중요도**(Authority)
* 데이터의 **최신성**(Freshness)
* 사용자 정의 필터(Custom Filters)

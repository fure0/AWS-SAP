## 🎙️ Amazon Polly란?

Amazon Polly는 **딥러닝** 기술을 사용하여 텍스트를 실제 사람의 목소리와 유사한 음성으로 변환(Text-to-Speech)하는 서비스입니다. Amazon Transcribe(음성→텍스트)와는 반대되는 개념입니다.

* **주요 특징:** 대화형 애플리케이션 제작 가능, 자연스러운 인간의 음성 제공.
* **엔진 타입:** 가장 자연스러운 음성을 생성하는 **신경망(Neural)** 엔진 등을 선택할 수 있습니다.

---

## 🛠️ 핵심 기능 1: 발음 어휘집 (Pronunciation Lexicons)

특정 단어의 발음을 사용자 정의할 때 사용합니다.

* **사용 목적:** 약어나 특이하게 표기된 단어를 원하는 방식으로 읽게 함.
* **예시:**
  * **약어 처리:** `AWS`라는 텍스트를 `Amazon Web Services`라고 풀어서 읽도록 설정.
  * **스타일링된 이름:** `st3ph4ne`처럼 숫자가 섞인 이름을 `Stephane(스테판)`으로 올바르게 발음하도록 교정.

* **방법:** 별도의 어휘집 파일을 생성하여 Polly에 업로드한 뒤, 음성 합성 작업 시 적용합니다.

---

## 🛠️ 핵심 기능 2: SSML (Speech Synthesis Markup Language)

음성 합성 마크업 언어를 사용하여 음성의 세부적인 스타일을 제어합니다.

* **사용 목적:** 단순 읽기를 넘어 음성의 뉘앙스와 흐름을 조절.
* **주요 기능:**
  * **강조:** 특정 단어나 구절을 강조함.
  * **정지(Pause):** `<break time="3s"/>`와 같은 태그를 사용하여 말하기 사이에 공백을 추가.
  * **특수 효과:** 숨소리 넣기, 속삭이는 소리(Whispering) 내기.
  * **말투 변경:** 뉴스 캐스터(Newscaster) 스타일 등 특정 상황에 맞는 어조 설정.
  * **발음 지정:** 소리 나는 대로 발음하도록 정밀 제어.

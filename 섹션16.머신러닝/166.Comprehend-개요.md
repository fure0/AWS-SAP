## ## 🧠 Amazon Comprehend 핵심 요약

Amazon Comprehend는 이름 그대로 텍스트를 **'이해(Comprehend)'**하는 서비스입니다. 기계 학습(ML) 전문 지식 없이도 텍스트에서 인사이트를 도출할 수 있는 **자연어 처리(NLP)** 서비스입니다.

### 1. 주요 특징

* **완전 관리형 및 서버리스:** 인프라 관리나 모델 훈련 없이 바로 API를 통해 사용할 수 있습니다.
* **비정형 데이터의 구조화:** 이메일, 고객 리뷰, 소셜 미디어 피드와 같은 비정형 데이터를 분석하여 의미 있는 특징을 추출합니다.

### 2. 주요 기능 (인사이트 추출)

Amazon Comprehend는 텍스트를 분석하여 다음과 같은 정보를 찾아냅니다.

* **언어 감지:** 텍스트가 어떤 언어로 작성되었는지 식별합니다.
* **엔티티 및 핵심 문구 추출:** 장소, 사람, 브랜드, 이벤트, 날짜 등을 추출합니다.
* **감정 분석 (Sentiment Analysis):** 텍스트가 긍정적인지, 부정적인지, 중립적인지, 혹은 혼합되어 있는지를 판별합니다.
* **구문 분석 (Syntax Analysis):** 토큰화 및 품사(형용사, 명사 등)를 사용하여 문장 구조를 분석합니다.
* **주제 모델링 (Topic Modeling):** 대량의 문서 그룹을 분석하여 공통된 주제(Topic)를 찾고 문서를 그룹화합니다.

## ## 💡 시험 문제 해결 포인트 (Cheat Sheet)

시험에서 다음과 같은 키워드가 나오면 **Amazon Comprehend**가 정답일 확률이 매우 높습니다.

| 상황 (Use Case) | 해결 방법 |
| --- | --- |
| **"자연어 처리(NLP)가 필요함"** | **Amazon Comprehend** 선택 |
| **"고객 이메일/리뷰의 긍정/부정 판단"** | **감정 분석** 기능 활용 |
| **"수만 개의 문서 내 주요 주제 자동 분류"** | **주제 모델링** 기능 활용 |
| **"텍스트에서 인명, 지명, 브랜드 추출"** | **엔티티 추출** 기능 활용 |

## ## 🛠 실제 활용 사례: 고객 경험 분석

1. **데이터 수집:** 고객 지원 이메일 및 티켓 데이터 수집.
2. **분석:** Comprehend를 통해 고객의 부정적인 피드백과 관련된 '핵심 문구' 추출.
3. **인사이트:** 특정 제품 기능에 대한 불만이 많음을 파악하고 비즈니스 개선 프로세스에 반영.

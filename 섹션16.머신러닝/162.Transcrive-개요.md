# 🎙️ Amazon Transcribe 핵심 정리

Amazon Transcribe는 **ASR(Automatic Speech Recognition, 자동 음성 인식)** 기술을 활용하여 음성 데이터를 텍스트로 변환해 주는 서비스입니다. 딥러닝을 통해 빠르고 정확한 변환을 지원합니다.

### 1. 주요 기능 및 특징

* **자동 음성 인식 (ASR):** 딥러닝 프로세스를 통해 음성을 실시간 또는 배치 방식으로 텍스트화합니다.
* **PII(개인 식별 정보) 제거 (Redaction):**
* 이름, 나이, 전화번호, 주민등록번호 등 민감한 정보를 자동으로 식별하여 가려줍니다(예: `[PII]`).
* 개인정보 보호가 중요한 고객 센터 데이터 처리에 유용합니다.


* **자동 언어 식별 (Language Identification):**
* 사용자가 말하는 언어를 스스로 판단합니다.
* 여러 언어(예: 영어, 프랑스어, 스페인어 등)가 섞여 있어도 똑똑하게 구분하여 텍스트로 변환합니다.



---

### 2. 주요 사용 사례

* **고객 서비스:** 고객 통화 내용을 대본으로 만들어 품질 관리 및 분석에 활용.
* **자막 생성:** 영상 콘텐츠의 자막을 자동으로 제작.
* **미디어 아카이빙:** 오디오/비디오 자산에 메타데이터를 부여하여 검색 가능한 형태로 저장.

---

### 3. 실습 요약 및 시연 결과

| 기능 | 입력 예시 (음성) | 변환 및 처리 결과 (텍스트) |
| --- | --- | --- |
| **기본 변환** | "안녕하세요, 이 코스가 정말 마음에 듭니다" | 즉시 정확하게 텍스트로 표시됨 |
| **PII 제거** | "제 이름은 스테판이고 31살이며, 번호는 910..." | 이름, 나이, 전화번호 부분이 **비식별 처리**됨 |
| **다국어 인식** | 영어와 프랑스어를 섞어서 발화 | 설정된 언어 후보군 내에서 자동으로 언어를 인식하여 변환 |

---

### 💡 요약 및 결론

Amazon Transcribe는 단순히 받아쓰기를 넘어, **보안(PII 제거)**과 **편의성(언어 자동 인식)**을 갖춘 강력한 음성 데이터 처리 도구입니다.

> **Note:** PII 제거 기능을 활성화하면 실시간 스트리밍 중에도 민감 정보를 안전하게 보호할 수 있습니다.

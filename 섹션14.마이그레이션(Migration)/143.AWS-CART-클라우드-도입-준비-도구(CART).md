## ☁️ AWS CART (Cloud Adoption Readiness Tool)란?

**AWS CART**는 조직이 클라우드 도입 및 마이그레이션을 진행할 때, 현재 얼마나 준비되어 있는지 스스로 평가하고 **효과적인 마이그레이션 계획을 수립할 수 있도록 돕는 자가 진단 도구**입니다.

### 1. 주요 목적

* **준비 상태 파악:** 클라우드 이전 작업을 AWS 모범 사례(Best Practices)에 맞춘 세부 계획으로 전환합니다.
* **가이드 제공:** 평가 결과를 바탕으로 부족한 부분을 어떻게 개선하고 구현해야 하는지 보고서를 통해 안내합니다.

### 2. 6가지 평가 관점 (Perspectives)

CART는 조직의 준비 상태를 다각도에서 분석하기 위해 다음 **6가지 영역**에 대한 질문을 제공합니다.

| 영역 | 설명 |
| --- | --- |
| **비즈니스 (Business)** | 클라우드 도입이 비즈니스 목표 및 가치와 일치하는가? |
| **사람 (People)** | 조직 내 인적 자원이 클라우드 기술을 수용할 준비가 되었는가? |
| **프로세스 (Process)** | 클라우드 운영을 지원하기 위한 워크플로우와 절차가 있는가? |
| **플랫폼 (Platform)** | 클라우드 아키텍처 및 기술적 기반이 준비되었는가? |
| **운영 (Operations)** | 클라우드 환경을 관리하고 모니터링할 역량이 있는가? |
| **보안 (Security)** | 데이터 보호 및 규정 준수를 위한 보안 체계가 마련되었는가? |

### 3. 결과 및 보고서

* **점수 차트:** 질문 답변을 마치면 각 영역별 점수가 포함된 차트가 생성됩니다.
* **준비도 보고서:** 마이그레이션 준비 정도를 시각적으로 보여주며, AWS 모범 사례를 적용하기 위한 구체적인 단계(Next Steps)를 제시합니다.

---

### 💡 시험 대비 팁 (SAP 시험)

AWS Certified Solutions Architect - Professional(SAP) 시험에서 CART는 아주 깊은 기술적 내용보다는 **"마이그레이션 초기 단계에서 조직의 준비 상태를 평가하는 도구"**라는 전반적인 개념 위주로 출제됩니다.

> **요약:** "비즈니스, 사람, 보안 등 6개 관점에서 질문을 던져 마이그레이션 준비 보고서를 뽑아주는 도구"라고 기억하세요!

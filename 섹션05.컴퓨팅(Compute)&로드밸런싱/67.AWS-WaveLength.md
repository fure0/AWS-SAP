## AWS Wavelength: 5G 네트워크를 위한 초저지연 인프라

AWS Wavelength는 통신 서비스 제공업체(CSP)의 **5G 네트워크 엣지**에 AWS 인프라를 구축하여, 모바일 기기 및 사용자에게 **밀리초(ms) 단위의 초저지연** 통신을 제공하는 서비스입니다.

### 1. 주요 개념 및 특징

* **Wavelength Zone:** 통신사의 5G 데이터 센터 내부에 설치된 AWS 인프라 배포 지점입니다.
* **5G 네트워크 통합:** 애플리케이션 트래픽이 통신사 네트워크를 벗어나 인터넷으로 나가지 않고도 AWS 서비스에 도달할 수 있습니다.
* **주요 배포 가능 자원:**
  * **Amazon EC2** 인스턴스
  * **Amazon EBS** 볼륨
  * **Amazon VPC** (Wavelength Zone으로 확장 가능)



### 2. 작동 원리 및 구조

* **통신사 게이트웨이(Carrier Gateway):** Wavelength Zone 내의 인스턴스가 5G 네트워크 및 인터넷과 통신할 수 있게 해주는 관문 역할을 합니다.
* **데이터 흐름:** 5G 사용자가 모바일 기기로 접속하면 트래픽이 5G 네트워크 내부의 Wavelength Zone으로 바로 전달됩니다. (인터넷을 거치지 않으므로 레이턴시가 극히 낮음)
* **상위 리전 연결:** Wavelength Zone에 배포된 EC2 인스턴스는 상위 AWS 리전(Parent Region)에 있는 **RDS**나 **DynamoDB** 같은 고가용성 서비스와 보안 접속을 통해 데이터를 주고받을 수 있습니다.

> **💡 핵심 팁:**
> AWS 자격증 시험이나 실무에서 **"5G"**와 **"초저지연(Ultra-low latency)"**이라는 키워드가 함께 등장한다면, 정답은 거의 대부분 **AWS Wavelength**입니다.

---

### 3. 주요 활용 사례 (Use Cases)

낮은 레이턴시가 필수적인 현대적인 애플리케이션에 주로 사용됩니다.

| 분야 | 상세 내용 |
| --- | --- |
| **스마트 시티** | 실시간 교통 제어, 공공 안전 모니터링 및 센서 데이터 처리 |
| **자율주행/커넥티드 카** | 차량 간 통신(V2X) 및 실시간 내비게이션 업데이트 |
| **미디어 & 엔터테인먼트** | 인터랙티브 라이브 비디오 스트리밍, AR/VR 콘텐츠 제공 |
| **게임** | 모바일 클라우드 게이밍 및 실시간 멀티플레이어 환경 |
| **의료** | 머신러닝(ML) 보조 진단 및 원격 의료 지원 |

---

### 4. 비용 및 계약

* **추가 요금 없음:** Wavelength 기능을 사용하기 위해 별도의 기본료나 초기 비용이 발생하지 않습니다.
* **서비스 계약:** 별도의 통신사 서비스 계약 없이 기존 AWS 계정을 통해 리소스를 생성하고 관리할 수 있습니다.
